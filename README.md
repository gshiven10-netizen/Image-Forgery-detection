# Image-Forgery-detection
An AI-powered web application for detecting image forgeries, including copy-move and splicing manipulations, using deep learning and computer vision techniques. The system highlights tampered regions and provides classification confidence with performance metrics.
🔍 Features:
- Detects Copy-Move Forgery
- Detects Image Splicing
- Highlights Manipulated Regions
- Displays Prediction Confidence
- Shows Model Accuracy & Confusion Matrix
- Clean and Interactive UI
